# Proveniência dos dados incluídos

## `delray_sqm_mean.csv`

Valores médios das leituras 5–8 apresentados por Hirama et al. (2022) para nove locais de Delray Beach, em 17 de agosto de 2020. Cada local possui duna, norte, oceano, sul e zênite. Unidade: magnitude por segundo de arco quadrado (`mag/arcsec²`). Menor valor significa horizonte mais claro.

Referência: DOI 10.1016/j.dib.2022.108430.

## Arquivos marcados como `demonstrative_assumption`

- `delray_beach_profile_demo.csv`;
- `delray_nests_demo.csv`;
- `obstacles_demo.csv`.

Esses valores foram criados apenas para permitir a execução do código. Não são levantamento de Delray Beach e não podem ser citados como dados de campo.

## Dados em `outputs_demo`

- `synthetic_observed_tracks.csv`;
- `synthetic_validation_tracks.csv`.

São simulações geradas com `synthetic_truth` do YAML. Servem para testar a calibração e a validação, não para inferência biológica.
